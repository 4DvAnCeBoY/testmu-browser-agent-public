# testmu-browser-agent

AI-native browser automation CLI and MCP server for Chrome.

## Install

```bash
npm install -g testmu-browser-agent
```

## Usage

```bash
testmu-browser-agent open https://example.com
testmu-browser-agent snapshot
testmu-browser-agent screenshot --output page.png
```

For full documentation, see the [main README](https://github.com/testmu/testmu-browser-agent).
